OOTP Fantasy Leagues: Release Notes

Product Release Information

Product: OOTP Fantasy Leagues
Release Number: 1.0.1
Release Date: Monday February 13, 2012
Support: For more information or support, please visit the OOTP Fantasy League Support Forum

Introduction

This document contains the release notes for OOTP Fantasy Leagues 1.0.1. The following sections describe the release in detail and provide late-breaking or other information that supplements the main documentation.

This is the final 1.0 production release. Users of previous versions of the mod are encouraged to upgrade their installation with the provided upgrade script.

Please see below for a list of bug fixes and enhancements found in this release. Your feedback is appreciated either through the official online forum, our online contact form or by filing a bug in the public bug submission tool..

What's New?

Enhancements

  • Added ability to uninstall fantasy database tables
  • Added OOTP Version field to install and options

Bug Fixes

  • Fixed bug where player state/territory of birth was resolving incorrectly for OOTP 12 games

Installation and Upgrade Notes

Upgrading
You may upgrade from version 1.0 Final usnig the provided upgrade script. To upgrade:.
  • Browse to http://[yoursiteURL.com]/[fantasy directory]/install/upgrade.php
  • Click the Upgrade Now button
  • Login and continue to use your league.
Manifest

Out of the Park Baseball
Version 10 or up (12 recommended)


Version Compatibility
This release is backwards compatible with version 1.0 Final only. Users of any previous versions must upgrade to version 1.0 or or use the full installation version.
