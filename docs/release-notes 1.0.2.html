OOTP Fantasy Leagues: Release Notes

Product Release Information

Product: OOTP Fantasy Leagues
Release Number: 1.0.2
Release Date: Sunday January 8, 2017
Support: For more information or support, please visit the OOTP Fantasy League Support Forum

Introduction

This document contains the release notes for OOTP Fantasy Leagues 1.0.2. The following sections describe the release in detail and provide late-breaking or other information that supplements the main documentation.

This is the final 1.0 production release. Users of previous versions of the mod are encouraged to upgrade their installation with the provided upgrade script.

Please see below for a list of bug fixes and enhancements found in this release. Your feedback is appreciated either through the official online forum, our online contact form or by filing a bug in the public bug submission tool..

What's New?

Enhancements

  • Updated support for OOTP 13+ database changes
  • Use of VORP changed to use of WAR for stats

Bug Fixes

  • Fixed error on player page "Unknown column 'logo_file' in 'field list'"
  • Fixed issues with updated images paths
  • Fixed bugs on player pages
  • Fixed display bugs on head to head page

Installation and Upgrade Notes

Upgrading
To upgrade, it is recommended to do a full uninstall and install a new copy of OOTP Fantasy Leagues.
